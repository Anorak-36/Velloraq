# Velloraq Branding

Velloraq is a coined product name for this project. It is short, pronounceable,
and intentionally not tied to one cloud provider or one serverless runtime.

Tagline:

> Self-hosted serverless security posture management for teams that need readable evidence, safe reports, and no proprietary lock-in.

CLI command:

```bash
velloraq scan
velloraq dashboard
```

The previous `slssec` command remains as a deprecated compatibility alias so
existing local workflows can migrate gradually.

## Name Availability

The project name is Velloraq, but legal uniqueness is not guaranteed by project documentation. Before a public or commercial release, maintainers should check GitHub, PyPI, Docker Hub, domain availability, trademark databases, and search engine results. Use a formal trademark review for legal certainty.

## Positioning

Velloraq is positioned as an open-source DevSecOps product for defensive,
authorized cloud and serverless audits. The naming avoids provider-specific
terms such as Lambda, Functions, or Cloud Run, leaving room for future support
across Kubernetes, IaC, policy-as-code, and compliance reporting.
